帮助中心

快速开始

1. 上传图片

支持三种方式上传包装图片：

  • • 点击"添加图片"按钮选择文件
  • • 直接拖拽图片到画布区域
  • • 使用 Ctrl+V / Cmd+V 粘贴剪贴板图片

2. 开始分析

上传图片后，系统会自动使用AI模型进行分析，检测包装设计中的问题。分析过程通常需要3-5秒。

分析完成后，检测到的问题会显示在右侧面板，并在图片上标注位置。

3. 查看结果

右侧面板会显示：

  • • AI检测的问题（带置信度评分）
  • • 提取的文字内容（OCR结果）
  • • 产品规格信息（自动提取）

4. QIL对比（可选）

如果有标准QIL文档，可以在右侧面板输入QIL信息，系统会自动对比并标出差异。

QIL使用指南

什么是QIL？

QIL（Quality Inspection List，质量检验清单）是产品包装的标准规格说明，包含产品名称、成分、规格、生产日期等信息。

如何输入QIL？

支持两种方式：

  • 1.
    文本输入：在右侧面板的QIL输入框中，按照"字段名: 值"的格式输入，每行一条。 
    产品名称: XX面霜
规格: 50g
保质期: 3年
  • 2.
    图片上传：点击"上传QIL图片"按钮，AI会自动识别图片中的表格信息。

QIL对比结果

系统会自动对比QIL与包装图片中的信息，结果分为三类：

  • • 匹配：QIL与图片信息一致
  • • 错误：QIL与图片信息不符
  • • 警告：部分匹配或需要人工确认

常见问题

分析准确率如何？

我们采用多步验证机制和行业规则引擎，综合准确率可达95%以上。每个检测结果都会提供置信度评分，您可以根据置信度判断可靠性。

支持哪些图片格式？

支持 JPG、JPEG、PNG、WebP 等常见图片格式。建议上传清晰度较高的图片以获得更好的分析效果。

数据安全吗？

所有数据传输采用HTTPS加密，云端存储使用Firebase服务，符合数据安全规范。免费版数据存储在本地浏览器，专业版数据加密存储在云端。

可以批量处理吗？

可以。免费版支持一次上传5张图片，专业版支持20张，企业版不限数量。系统会自动并发处理，大幅提升效率。

如何切换AI模型？

专业版用户在图片卡片上方可以看到模型选择器，点击即可切换。切换后可以重新分析图片以获得不同模型的结果。

遇到问题怎么办？

您可以通过邮件 support@packverify.com 联系我们，专业版用户享有优先支持，通常24小时内响应。
